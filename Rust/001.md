数据存放在 heap 和 stack 的区别:

- stack: fast, fix-size 寿命确定
  - "hello"
  - let arr = [1, 2, 3]
- heap: slow, mutable 大小不确定, 长期储存
  - String::from("hello") 移动所有权(不是复制), 用于返回
  -  vec![1, 2, 3]

&str 不可变借用

`=`, copy 和 clone 的区别:

- copy trait: `=` 就是 copy, 在 stack 上, 不会移动所有权
- none copy trait: `=` will move ownership
- clone 分配内存
- String 不能 copy, 只能 move 或者 clone

java 和 rust 中 `==` 的区别:

- java: `==` identity check, `.equals()` value check
- rust: PartialEq, similiar to `.equals()`

iter_mut() 和 &mut: 

- iter_mut() 是逐个借用, 超短生命周期

ref == 在 pattern 里写 & 的等价物
